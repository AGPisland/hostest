AG365 INFORMATICA

AG365 INFORMATICA es un sistema ETL empresarial de última generación que proporciona una base estable para su entorno de nube híbrido.

Lo más destacado:

  • Extraccion de datos API REST.
  • Implementacion sobre servicios en nube SaaS.
  • Modelo de negocio en base a la cantidad de ejecuciones lambda aws solicitada por el uso de AG365.
